# FPS ERPNext

**Fast Planet Shipping LLC** — Custom ERPNext app for freight forwarding operations.

## Overview

This app extends ERPNext for **Fast Planet Shipping LLC**, a UAE-based freight forwarding company. It adds:

- **FPS Workspace** — Dashboard with KPIs and quick actions, appearing as a top-level tile on `/desk`
- **FPS Enquiry** module — Capture incoming customer inquiries
- **Job Order** module — Manage freight shipments end-to-end
- **Customs Tracker** / **Daily Job Tracker** — Operations tracking
- **Proof of Delivery** — Delivery confirmation workflow
- **AR/AP Charges** — Profit calculation per job
- **UAE-specific** chart of accounts, VAT 5%, cost centers
- **HR** entitlements per UAE Labour Law

## Installation on Frappe Cloud (Private Bench)

1. Go to your **Bench dashboard** on Frappe Cloud
2. Click **Apps → + Add App → Install from GitHub**
3. Repository URL: `https://github.com/YOUR_USERNAME/fps_erpnext`
4. Branch: `main`
5. Click **Install**
6. Deploy the bench
7. From **Sites** → your site → **Apps** → click **Install** next to `fps_erpnext`

## Tech Stack

- **Frappe Framework v16**
- **ERPNext v16**
- **Python 3.10+**
